Introduction

Hal.js is a stable and powerful hardware enumerator with battery level checker, right in the web browser.

Installation

// Using npm
npm install systeminformation

// Using yarn
yarn add systeminformation

About

To use you need a Chromium based web browser.

  • Stable client
  • support typescript
  • 100% Compatible with Lavalink
  • Object-oriented
  • 100% Customizable
  • Easy to setup
  • Inbuilt mobile phone support
